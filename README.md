# Prototipo visual de TrueFocus

Abre [index.html](./index.html) en el navegador para ver la galería de pantallas.

Incluye:

- Marco de teléfono tipo iPhone para presentación.
- Modo claro, modo oscuro o ambos.
- Cambio de color principal entre morado, azul, verde, naranja, rosa y rojo.
- Pantallas principales de la app: bienvenida, login, permisos, protección activa, inicio, enfoque, agenda, apps bloqueadas, racha, ajustes, editar perfil y overlay de bloqueo.

Está pensado para mostrar el diseño sin depender de Android Studio ni del APK.
